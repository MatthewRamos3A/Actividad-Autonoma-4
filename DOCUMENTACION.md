# Optimización de Código y Medición de Tiempos

## Introducción

En esta actividad se trabajó con un programa en Python para encontrar números primos dentro de un rango determinado. Primero se ejecutó un código normal y después se realizaron mejoras para reducir el tiempo de ejecución.

## Optimización

Las mejoras aplicadas fueron:

- Uso de la raíz cuadrada para disminuir iteraciones.
- Comprensión de listas.
- Uso de NumPy.
- Mejor organización del código.

## Resultados

Código original: 37.44 segundos

Código optimizado: 0.33 segundos

Se observó una mejora importante en la velocidad del programa después de aplicar las optimizaciones.

## Conclusiones

Optimizar un programa ayuda a disminuir tiempos de ejecución y aprovechar mejor los recursos del sistema.
